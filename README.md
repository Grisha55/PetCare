# 🐾 PetCare — трекинг прививок и ухода за питомцами

PetCare — это frontend-приложение для владельцев домашних животных, которое помогает удобно хранить медицинскую информацию о питомце и никогда не забывать о прививках, приёме лекарств и других важных процедурах.

Цель проекта — заменить бумажный ветеринарный паспорт удобным цифровым решением, доступным в любое время с телефона или компьютера.

## ✨ Основные возможности

- 🐶 **Профиль питомца** (фото, имя, порода, дата рождения)
- 💉 **Учёт прививок** с датами и статусами
- 💊 **Учёт приёма лекарств** и прививок
- ⏰ **Напоминания** о предстоящих процедурах
- 📄 **Хранение документов** (ветпаспорт, справки)
- 🧠 **Раздел с полезными советами** по уходу
- 👤 **Профиль пользователя** и настройки

## 🎯 Для кого это приложение

- Владельцы собак и кошек
- Люди, которые хотят заботиться о питомце системно
- Те, кому неудобно хранить информацию только в бумажном паспорте
- Пользователи, ценящие простой, милый и понятный интерфейс


## 🛠️ Технологический стек

**Frontend**
- React
- TypeScript
- Vite
- SCSS Modules
- Context API (глобальное состояние)

## 📐 Архитектура проекта

Проект построен с использованием Feature-Sliced Design (FSD) — архитектурного подхода к frontend-разработке, ориентированного на бизнес-домены и пользовательские сценарии.

Архитектура реализована в облегчённом виде, без избыточного усложнения, что позволяет быстро развивать продукт и при этом сохранять возможность масштабирования.


### ⚠️ Примечания по архитектуре

- Используется упрощённая версия FSD
- Слой processes намеренно не применяется
- Redux не используется — для MVP выбран Context API
- Архитектура изначально спроектирована с возможностью роста
